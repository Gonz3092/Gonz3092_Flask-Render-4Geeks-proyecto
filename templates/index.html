API de Predicción de Diabetes

Esta API permite predecir la probabilidad de diabetes basándose en ciertos parámetros médicos.

Endpoint de Predicción

POST /predict

Parámetros de la solicitud (JSON):

  • Pregnancies: Número de embarazos (numérico)
  • Glucose: Concentración de glucosa (numérico)
  • BloodPressure: Presión arterial (numérico)
  • SkinThickness: Grosor del pliegue cutáneo (numérico)
  • Insulin: Insulina sérica (numérico)
  • BMI: Índice de masa corporal (numérico)
  • DiabetesPedigreeFunction: Función de pedigrí de diabetes (numérico)
  • Age: Edad del paciente (numérico)

Ejemplo de Solicitud JSON:


{
    "Pregnancies": 6,
    "Glucose": 148,
    "BloodPressure": 72,
    "SkinThickness": 35,
    "Insulin": 0,
    "BMI": 33.6,
    "DiabetesPedigreeFunction": 0.627,
    "Age": 50
}

Ejemplo de Respuesta JSON:


{
    "prediction": 1,
    "prediction_label": "Positivo en Diabetes",
    "probability_negative": 0.2,
    "probability_positive": 0.8
}
